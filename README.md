# Arduino RGB LED strip controller
This project lets user controle LED RGB strip (NeoPixel) to get controlled by IR remote.

---

## List of used parts
#| Part name | Quantity
--- | --- | ---
1 | Arduino Uno | 1
2 | Breadboard | 1
3 | IR receiver | 1
4 | IR remote | 1
